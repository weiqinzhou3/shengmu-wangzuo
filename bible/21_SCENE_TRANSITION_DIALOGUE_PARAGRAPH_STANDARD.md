# 二十一、场景转移、重要对话与段落排版标准

> 本文件依据第二、三章人工审阅建立。正文不能只保证单句成立，还必须保证场景之间、人物知识之间和段落视觉之间连续。

## 1. 场景转移必须完整

当人物从一个地点移动到另一个地点时，正文原则上必须交代以下内容：

1. 人物为什么离开当前地点；
2. 人物通过什么方向、路径或方法移动；
3. 移动途中至少一个影响判断或身体状态的变化；
4. 到达新地点后最先看见、听见或确认的事物；
5. 人物如何从新证据得出下一步行动，而不是旁白提前宣布结论。

禁止以下跳切：

```text
人物熄灭篝火。
下一段直接出现数里外的车辙、城门或建筑。
```

应写清“离开篝火—绕行营地—到达北门—筛选痕迹—确认路线”的连续过程。场景转移不是无价值的路程描写；它必须承担方向确认、时间流逝、体力消耗、风险变化或人物决策中的至少一项功能。

## 2. 重要对话必须先校验知识边界

在写重要对话前，必须先回答：

- 角色A亲眼看见了什么；
- 角色B亲眼看见了什么；
- 谁先离开现场；
- 谁能够知道后续事件；
- 双方是否已经交换姓名、身份和所属关系；
- 哪些内容是事实，哪些只是推断。

禁止人物询问对方不可能知道的情况。例如瓦蕾莎先被押离接应营，就不能在获救后向更晚离开、随后坠入涵洞的沈砚询问完整营地结局，并把对方的猜测当作事实。

禁止角色无来源地知道另一人的姓名。姓名必须来自：

- 当面自我介绍；
- 明确听见他人称呼；
- 亲眼阅读登记文件；
- 已在正文中出现的可靠介绍。

如果来源未在正文建立，角色只能使用身份称呼，例如“华夏医官”“工程师”“护送官”。

## 3. 重要对话必须有议题结构

承担世界观揭示、关系转折或谜团解答的对话，必须具有明确议题，而不是想到什么问什么。

推荐顺序：

1. 当前必须解决的问题；
2. 双方能够确认的共同事实；
3. 各自掌握但对方不知道的信息；
4. 仍然无法确认的未知项；
5. 信息如何改变双方下一步选择；
6. 关系边界或合作条件。

对话应通过动作、伤势处理、物证、环境和停顿自然推进。不能连续使用十余轮“问一句、答一句”的审讯式结构，也不能把设定资料拆成互不关联的台词。

角色必须区分事实与推断。瓦蕾莎可以确认原始军令、灰封车去向异常、续界石签章和白誓机制，但不能在没有账簿与证人的情况下直接宣布完整走私网络。沈砚可以形成强推断，但不能把推断写成作者全知结论。

## 4. 正常叙事段落排版

“一句一段”只能用于极少数真正需要停顿的落点，不能成为整章默认排版。

正常叙事段落应以二至五句为主，使同一动作、感官、因果或判断保持在同一个语义单元内。以下情况应合并在同一段落：

- 一个动作及其身体反馈；
- 环境变化及人物判断；
- 一句台词及说话者同时发生的动作；
- 前因、动作和直接后果；
- 同一物件的观察与由此产生的推断。

连续三个以上单句短段落通常意味着正文正在退化为分镜、提纲或手机聊天式排版，必须重新检查。

允许单独成段的内容包括：

- 章节转折点；
- 无法被其他句子替代的关键决定；
- 文书、碑文、军令和仪式文字；
- 极少量需要形成视觉停顿的结果句。

即使使用短段落，也不得连续承担完整动作过程。

## 5. 章节长度与拆分

当一章同时包含以下两个或更多完整戏剧单元时，应优先拆章，而不是继续堆叠：

- 长距离追踪或场景迁移；
- 潜入与营救；
- 战斗或逃脱；
- 伤势处理；
- 大规模世界观揭示；
- 关系与合作条件改变。

拆章必须以叙事目标区分，而不是在任意字数位置截断。例如：

- 第三章负责“找到并进入祭坛”；
- 第四章负责“中断仪式、逃离并交换关键信息”。

每章结尾应完成本章目标的一部分，同时留下下一章明确且自然的问题。

## 6. 提交前硬性检查

- 是否从上一场景自然移动到下一场景；
- 是否出现人物突然知道姓名、地点或后续事件；
- 是否让先离场的人询问后离场者无法知道的事实；
- 重要对话是否有明确议题和结论；
- 是否区分事实、合理推断和未知项；
- 是否连续出现三个以上单句短段落；
- 是否为了压缩篇幅省略了必要的动作、距离、阻力或承上启下文字；
- 章节是否包含过多完整戏剧单元，应当拆分。
