# 十一、最高优先级硬规则

以下规则正式锁定。修改必须明确升级小说圣经版本。

1. 英雄大陆是真实异世界，不是游戏或虚拟空间。
2. 现实国家通过文明之门介入，但不能携带枪械、通讯设备、电力设备和现代高科技体系。
3. 外来者的主要优势是知识、方法、组织与人才，不是现代装备。
4. 外来者默认不能直接施法；沈砚是极少数法则授印例外。
5. 不使用完整全视之眼、属性面板、忠诚度和战斗胜率。
6. 保留一至七阶兵种分类，但阶位是军事社会分类，不是系统数值。
7. 英雄技能是学科、经验和领域，不是技能点。
8. 所有资源都是实物，存在运输、储存、加工与损耗。
9. 所有城市人口、兵力和建筑都有真实来源与时间成本。
10. 兵营、墓地、工坊和巢穴不凭空生成生命。
11. 丧钟城资源贫瘠，没有稳定高阶兵种巢穴。
12. 丧钟城的高阶人物主要通过救援、外交、投奔、契约和共同经历加入。
13. 无名钟域仅用于本土防御和身份稳定，不能对外横推。
14. 丧钟城是无乡者共同体，不是普通亡灵帝国。
15. 沈砚始终是牧师和领主，不转为传统亡灵法师。
16. 瓦蕾莎在早期加入，是第一代核心团队成员。
17. 白骨长峡中第一代核心团队除沈砚外全部死亡。
18. 沈砚只成功拉回瓦蕾莎；其余成员永久不可复活。
19. 瓦蕾莎成为唯一的八阶堕落天使。
20. 堕落天使无法量产，不能通过建筑、资源或相同仪式复制。
21. 当前只锁定瓦蕾莎这一名真正八阶；全书真八阶总数不超过三，伪八阶不超过一。
22. 八阶不是完整科技树，不覆盖所有七阶兵种。
23. 遗愿不等于完整灵魂，亡灵化不等于复活。
24. 开智兵种一旦形成连续人格，就作为独立人物书写，拥有姓名、选择、情感与政治立场。
25. 开智人物不得被无意义消耗；死亡必须有因果、选择和长期影响。
26. 国家和联盟都不能整体降智或绝对善恶。
27. 后期战争是现实国家与异世界政体交叉结盟后的全面战争，不是现实世界对异世界。
28. 主角不统一整个大陆，最终结果是二十年武装停战。
29. 每项政治议题必须通过案件、建设、探索、人物或战争呈现。
30. 每次胜利必须改变后续格局并产生代价。
31. 采用群像叙事，重要人物拥有脱离主角的目标、关系和行动。
32. 任何重大历史事件必须具备完整动机、背景、信息条件、资源限制和可追溯后果。
33. 公开正文中的现实国家统一使用小说代称，不直接出现现实国家正式名称或现实政治人物姓名。
34. 太阳国是重要现实政体之一，具有内部派系、独立利益与复杂人物，不得整体脸谱化。
35. 本书包含爱情线，但不写后宫；爱情不得取消人物独立性或替代法则因果。
36. 总体风格为史诗骨架、人物驱动、严肃底色、克制幽默；不得直接模仿在世作者的具体文风。
37. 新人物、新事件、新背景、新兵种、新政体与重大关系变化必须在同一版本中回写对应设定和状态文件。
38. 用户为读者与方向裁决者，Codex为主要作者并负责正文、设定回写和机械一致性；Reviewer由用户另行指定，作者不得自行替代最终阅读裁决。
39. 宝物采用D、C、B、A、S五级分类，但不显示数值面板；S级是文明法则遗产，不等于单纯战力最高。
40. 所有重要宝物必须有实体、来源、法律所有者、当前持有者、保管者、所在地、维护要求和使用代价。
41. 宝物不会因击杀自动掉落，也不能无视血脉、信仰、魔力、契约和共鸣限制。
42. 正文中宝物首次出现、转移、损坏或发现新效果时，必须同步更新 `state/ARTIFACT_LEDGER.md`。
43. 英雄大陆全球地理采用类似现实世界的拓扑关系以降低理解成本，但海岸线、边界、文明和历史必须保持原创。
44. 丧钟城位于苍穹主陆中央偏东的灰脊边境，资源贫瘠但处于多文明交通节点；白骨长峡必须在大战前长期铺垫。
45. 远距离移动、战争和贸易必须遵守地图、道路、季节、补给、交通工具和通讯限制。
46. 每个国家、联盟和重要政体必须拥有背景、物质基础、内部派系、核心目标、安全焦虑和红线。
47. 国家或联盟的大战、结盟、背叛、政变不得只为剧情高潮服务，必须先登记完整行动动机。
48. 在线 GitHub 仓库 `weiqinzhou3/shengmu-wangzuo` 是唯一正式事实来源；新增设定与正文通过独立分支、提交、外部Review和Pull Request合并。
