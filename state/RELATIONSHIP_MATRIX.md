# 第一卷后续关系计划

## 沈砚—瓦蕾莎

### D2：鲁尔选择

- 瓦蕾莎第一次看到低阶骷髅公开拒绝原主人
- 她不承认鲁尔与活人相同，但接受“不能立即摧毁”的最低判断

- 沈砚不要求瓦蕾莎无限净化水源，而是限制她只处理急救饮水
- 瓦蕾莎第一次相信沈砚不会把她作为高阶工具耗尽
- 两人建立工作层面的信任，不产生恋爱确认

### D14—D16：守城

- 瓦蕾莎负责混合卫队最低限度训练，但不宣誓私人效忠
- 沈砚给予撤离选择，不以救命债要求她留下
- 瓦蕾莎在可以离开的情况下主动留守，关系升级为战场信任

### D20：成为城民

- 瓦蕾莎宣誓对象是丧钟城公约，而非沈砚个人
- 沈砚接受她拥有独立政治身份

## 爱情线审查规则

- 救命之恩不自动转化为爱情
- 关系跃迁必须有共同事件、长期互动或明确选择
- 爱情对象保留政治立场、职业责任和拒绝权
- 不以重复误会维持暧昧
- 重要感情变化同步更新人物状态与记忆包
