# 第一卷第一次守城战力基线

> 本文件是第一卷守城战的卷级数量上限与战术边界。精确姓名、伤亡和分场投入在逐章大纲与正文中更新。

## 战役性质

- 类型：边境城镇级冲突
- 地点：破败的丧钟城
- 结果目标：守住居民、黑色钟塔、粮仓核心区和证据；迫使地方主体谈判
- 非目标：歼灭敌军、击败正规军团、取得完整主权承认

## 丧钟城

| 类别 | 数量区间 | 作用 |
|---|---:|---|
| 正式编组战斗者 | 20—24 | 守点、巡逻、局部反击 |
| 辅助防守者 | 12—16 | 运输、灭火、救护、投石、报警、封门 |
| 低智亡灵工程与障碍队 | 20—26具 | 搬运、挖掘、堵路、固定障碍，不参与政治授权 |
| 战场活动实体 | 55—65 | 不等于全员直接战斗 |

### 限制

- 无正规兵营和攻城器械
- 军械不足且规格混乱
- 外墙无法完整防守，只能收缩核心区域
- 瓦蕾莎不能飞行，不能连续使用高强度圣光
- 沈砚主要承担组织、救治、谈判和局部指挥

## 圣堂地方执法队

| 类别 | 数量区间 |
|---|---:|
| 总人数 | 32—40 |
| 正式战斗人员 | 24—30 |
| 牧师、书记官、车夫、看守、证物人员 | 8—10 |

### 限制与目标

- 无大型攻城器械
- 无七阶单位
- 需要活捉瓦蕾莎并取得接应营记录
- 不愿彻底烧毁证据和城市
- 地方指挥链不愿因一次秘密行动承担过高伤亡

## 墓园收尸商队

| 类别 | 数量区间 |
|---|---:|
| 总实体 | 28—36 |
| 商人、施法者与护卫 | 8—11 |
| 受控骷髅与行尸 | 16—22具 |
| 车夫与搬运人员 | 4—6 |

### 限制与目标

- 希望完整取得争议亡灵、尸骸和旧城档案
- 不愿让圣堂焚毁目标
- 商业武装缺少持续攻坚能力
- 受控亡灵在无名钟域附近会出现控制延迟

## 边境匪团

| 类别 | 数量区间 |
|---|---:|
| 总人数 | 42—55 |
| 愿意直接投入战斗 | 28—36 |
| 侦察、车夫、驮兽看守与搬运 | 14—19 |
| 精锐核心 | 8—10 |

### 限制与目标

- 主要目标为粮食、工具和可转卖物品
- 缺少攻城装备和长期补给
- 需要保护退路、驮兽和赃物运输
- 伤亡明显上升后会撤退

## 整体压力

- 三方总来袭规模：105—130人或亡灵实体
- 同一时刻直接投入战斗：45—65
- 三方到达时间、目标和攻击方向不同
- 没有统一指挥
- 各方均被粮仓代理人的信息误导

## 鲁尔的战役职责

鲁尔不以个人击杀数证明价值，而承担：

- 判断残墙、墓道和采石结构的承重状态
- 组织低智亡灵夜间施工
- 设计可放弃的外层障碍与内层阻断点
- 在沈砚不在场时独立决定封堵路线
- 证明觉醒亡灵能够主动保护活人赖以生存的城市

该职责直接引出第二卷的石工、工程队、亡灵劳动制度与城市建设线。
