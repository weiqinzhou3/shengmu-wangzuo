# 事件账本

## 文明之门开启

- 状态：背景事件，具体时间待锁定
- 直接原因：未知
- 结构性原因：两个世界法则开始连接
- 主要参与者：现实各国、英雄大陆邻近政体
- 长期影响：现实国家开始向英雄大陆派遣人员并建立合作
- 世界真相：暂不公开

## 丧钟城城主承认

- 状态：计划
- 直接原因：丧钟城无主且面临生存与外部吞并
- 发起者：沈砚、丧钟城居民、城市核心
- 核心条件：核心承认、居民承认、外部政体承认
- 长期影响：沈砚成为无乡者之主

## 白骨长峡围猎

- 状态：远期计划
- 直接原因：营救被俘流亡者和失联人员
- 结构性原因：丧钟城的人口与人格制度威胁多个旧秩序
- 参与者：丧钟城第一代核心团队、圣堂审判庭、墓园奴役派、地下城部分贵族、现实开拓集团
- 结果锁定：第一代核心团队除沈砚外全灭；瓦蕾莎异常返生
- 长期影响：八阶出现，丧钟城进入新政治时代
