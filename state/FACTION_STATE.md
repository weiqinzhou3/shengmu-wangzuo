# 国家、联盟与政体动态状态

> 本文件记录当前时间点的行动状态。长期背景见 `bible/18_FACTION_DOSSIERS.md`。

## 正文开始前总体状态

- 现实国家已经发现文明之门，但公开信息与进入规模有限。
- 英雄大陆各联盟只知道少量“无魔之界来客”，判断分歧巨大。
- 丧钟城仍是被遗忘的贫瘠弃城，不是区域性政体。
- 卷一中的圣堂、墓园与匪团行动都属于地方主体，不代表整个联盟。

# 重大行动登记

## 行动A：灰髓疫净化与灭证

- 发起势力：圣堂边境地方指挥体系
- 直接目标：隔离结界失效前消除疫病与难民队伍
- 结构性利益：维持边境秩序、信仰净化权和地方指挥者职位
- 内部支持者：强硬审判人员、参与非法运输者
- 内部反对者：部分普通骑士、程序型牧师、华夏先遣人员
- 掌握信息：确有两例灰髓疫；隔离结界两小时内失效
- 错误或被操纵认知：把整个难民队伍都视为感染者；低估华夏独立记录能力
- 可用资源：营地执法队、圣光焚化设施、封锁结界
- 时间窗口：D0两小时
- 替代方案：转移至石窟隔离站、分区隔离、请求邻近教区支援
- 未采用原因：所需驮兽与护卫已被秘密调去运输未申报星银和战俘；申请支援会暴露
- 失败成本：疫病外泄、走私曝光、瓦蕾莎把记忆带回天界
- 实际结果计划：焚烧未完全执行；记录外泄；瓦蕾莎抗命；沈砚失联

## 行动B：断翼封名礼

- 发起势力：圣堂地方审判者
- 直接目标：剥夺瓦蕾莎军职、天界锚点与证词能力
- 结构性利益：阻止她死亡后回归天界并暴露D0真相
- 支持者：参与灭证的审判人员
- 反对者：尚未获知完整情况的上级教区、部分普通骑士
- 掌握信息：瓦蕾莎公开抗命并目睹非法名单
- 可用资源：第七静默祭坛、白誓、地方惩戒权限
- 时间窗口：七十二小时
- 替代方案：监禁、押返、直接处决
- 未采用原因：监禁与押返会触发上级复核；直接死亡可能回归天界
- 失败成本：瓦蕾莎存活并公开证言
- 实际结果计划：沈砚D1中断仪式

## 行动C：三方交割窗口

- 发起者：丧钟城粮仓代理人
- 直接目标：迫使城市投降，并让多个买家竞价保护其利益
- 结构性利益：维持水粮垄断、避免人格审判追责
- 信息来源：粮仓、离城名单、瓦蕾莎公开活动、鲁尔争议登记
- 可用资源：商路联系人、信使、伪造城门安排
- 时间窗口：D12发信，D14—D15抵达
- 三份信息：
  - 圣堂：D15中午开门，交出瓦蕾莎与证据
  - 墓园：D15傍晚西门交割争议亡灵
  - 匪团：D15夜间抢粮
- 错误认知：认为三方会按时间分批行动，且自己能在投降后继续控制粮仓
- 失败成本：三方互相发现欺骗，城内垄断网络暴露
- 实际结果计划：三方在36小时内先后到达，彼此目标冲突被沈砚利用

## 行动D：圣堂边境执法队抵城

- 发起势力：圣堂边境副裁判官所属机构
- 直接目标：带回瓦蕾莎、接应营证据和地方逃亡人员
- 内部支持者：地方强硬派
- 内部反对者：不愿为隐瞒事件承担大量伤亡的普通军官
- 掌握信息：瓦蕾莎在城内；有人承诺开门
- 错误认知：丧钟城居民会因水粮压力交出异族
- 可用资源：有限执法队、少量骑乘、正式印信
- 替代方案：向上级教区申请大军或公开审理
- 未采用原因：会暴露D0处置与地方越权
- 失败成本：证据上交、地方指挥链被调查
- 结果计划：签署30日《灰脊停火与移交暂缓书》

## 行动E：墓园收尸商抵城

- 发起势力：持牌收尸代理人
- 直接目标：保全索赔权、夺回鲁尔等争议个体与尸骸
- 掌握信息：城内存在逃逸亡灵，钟域未完全恢复
- 错误认知：圣堂不会同时出现；城市没有统一法律
- 可用资源：商队护卫、低智亡灵、商业许可与索赔文书
- 替代方案：向墓园上级领主提起正式诉讼
- 未采用原因：周期长，争议亡灵可能消失或被圣堂净化
- 失败成本：许可、货物与市场信用
- 结果计划：签署30日《遗骸与人格争议保全契》

## 行动F：边境匪团抢掠

- 直接目标：抢粮与可转卖宝物
- 信息：守军不足、粮仓位置、城门即将开放
- 错误认知：未预料圣堂和墓园同时在场，也不知道钟域会干扰亡灵控制
- 可用资源：轻装人员、夜袭经验
- 替代方案：等待围城结束后捡拾残余
- 未采用原因：担心其他势力先拿走粮食
- 失败成本：伤亡与退路被封
- 结果计划：未签政治文书，换取重伤者基础救治后撤离

# 外部承认边界

卷一结束时：

- 圣堂文书只约束一个地方执法体系30日
- 墓园文书只约束一个持牌收尸商及本次索赔30日
- 匪团不承认丧钟城政治地位
- 丧钟城获得的是“能够缔约与保管争议”的事实地位，不是完整主权承认
