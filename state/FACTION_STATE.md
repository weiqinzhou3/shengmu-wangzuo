# 国家、联盟与政体动态状态

> 本文件记录当前时间点的行动状态。长期背景见 `bible/18_FACTION_DOSSIERS.md`。

## 正文开始前总体状态

- 现实国家已经发现文明之门，但公开信息与进入规模有限。
- 英雄大陆各联盟只知道少量“无魔之界来客”，判断分歧巨大。
- 丧钟城仍是被遗忘的贫瘠弃城，不是区域性政体。
- 卷一中的圣堂、墓园与匪团行动都属于地方主体，不代表整个联盟。

# 重大行动登记

## 行动A：灰髓疫净化与灭证

- 发起势力：圣堂边境地方指挥体系
- 直接目标：隔离结界失效前消除疫病、难民证人和记录
- 结构性利益：维持地方指挥权、净化裁量权并掩盖非法星银与战俘运输
- 内部支持者：强硬审判人员、参与非法运输者
- 内部反对者：部分普通骑士、程序派牧师、华夏先遣人员
- 掌握信息：确有两例灰髓疫；结界两小时内失效；三级转运仍可通过人力灰封车和低智骷髅启动
- 被操纵认知：把整个难民队伍都列为感染者；宣称驮兽不足即等于完全无法转运
- 可用资源：执法队、焚化设施、四辆人力灰封车、十二具低智骷髅、旧白灰窑两枚续界石
- 标准替代方案：三十分钟装载 → 一小时内进入旧白灰窑 → 续界八小时 → 五至六小时到石窟隔离站
- 未采用原因：标准流程会暴露被挪用的驮兽、封车、护卫和未申报货物
- 失败成本：疫病外泄、走私曝光、瓦蕾莎向上级提交证言
- 结果：焚烧未完全执行；记录外泄；瓦蕾莎抗命；沈砚失联

## 行动B：秘密扣押瓦蕾莎

- 发起势力：圣堂地方审判者
- 直接目标：把瓦蕾莎移出公开程序，在正式复核前令其失去行动与作证能力
- 结构性利益：阻止其携带D0证据进入上级教区的正式审查
- 支持者：参与灭证的审判人员
- 反对者：尚未知情的上级教区、部分普通骑士
- 可用资源：第七静默处置点、四名营地审判执事、两名处置点看守、更高层普通书面授权
- 时间窗口：正式审查人员抵达前
- 替代方案：营地公开监禁、立即押返、依法治疗后复核
- 未采用原因：公开程序会留下完整名册、伤情和命令记录
- 失败成本：瓦蕾莎存活并公开证言
- 结果：沈砚D1救出瓦蕾莎并带走扣押记录

## 行动C：三方交割窗口

- 发起者：丧钟城粮仓代理人
- 直接目标：促成投降并让多个买家竞价保护其利益
- 结构性利益：维持水粮垄断、避免人格审判追责
- 信息来源：粮仓、离城名单、瓦蕾莎公开活动、鲁尔争议登记
- 可用资源：商路联系人、信使、复制路证和伪造城门安排
- 三份信息：
  - 圣堂：D15中午开门，交出瓦蕾莎与证据
  - 墓园：D15傍晚西门交割争议亡灵
  - 匪团：D15夜间抢粮
- 错误认知：认为三方会分批行动，且自己能在投降后继续控制粮仓
- 失败成本：三方发现欺骗，垄断网络与杀人灭口事实暴露
- 结果：三方在36小时内先后到达，彼此目标冲突被丧钟城利用

## 行动D：丧钟城多源预警与战时授权

- 发起者：塞西娅、鲁尔、灰雁商队与城外观察人员分别提供信息
- 直接目标：在不掌握三封信全文的情况下核实外部武装正在接近
- 信息来源：圣堂旗号与人数观察、重复路证与封蜡、墓园车辙与牌照、受伤信使的D15路线筹码
- 核验结果：至少两支有组织武装和一支南侧轻装队将在三日内抵达
- 信息限制：不知道各方全部条件，也不知道粮仓代理人承诺的精确开门方式
- 对外保密：情报只在临时议事结构和守城负责人中按需分发，公开内容只包括可核实的移动事实
- 结果：D12居民公开表决授予沈砚有限战时看守权

## 行动E：圣堂地方执法队抵城

- 直接目标：带回瓦蕾莎、接应营证据和地方逃亡人员
- 人数区间：24—30人，其中18—22名可直接战斗
- 限制：没有大型攻城器械，必须尽量活捉瓦蕾莎并取得记录
- 错误认知：居民会因水粮压力按约开门
- 替代方案：向上级申请公开审理或大规模支援
- 未采用原因：会暴露D0越权和非法运输
- 失败成本：地方指挥链被调查
- 结果：签署30日《灰脊停火与移交暂缓书》

## 行动F：墓园收尸队抵城

- 直接目标：保全索赔权、取得鲁尔等争议个体与尸骸
- 人数区间：16—22人，其中6—8名人员及8—12具受控骷髅
- 限制：希望资产、档案和尸骸保持完整；钟域削弱远程控制
- 错误认知：圣堂不会同时出现，城市没有统一法律
- 替代方案：向墓园上级法庭起诉
- 未采用原因：周期长，争议亡灵可能被圣堂净化或转移
- 失败成本：许可、货物与市场信用
- 结果：签署30日《遗骸与人格争议保全契》

## 行动G：边境匪团抢掠

- 直接目标：抢粮与可转卖物资
- 人数区间：30—38人，实际投入巷战者18—24人
- 限制：缺少攻城器械，需留人看守驮兽、退路和赃物
- 错误认知：未预料圣堂与墓园同时出现，也不知道外仓是诱饵
- 替代方案：等待围城结束后捡拾残余
- 未采用原因：担心其他势力先拿走粮食
- 结果：不签政治文书，重伤者获得基础救治后撤离

## 行动H：灰雁商队证据保管

- 直接目标：让道路恢复最低稳定，避免货物和商路被单方吞并
- 商队规模：约11人、5辆车和若干驮兽
- 可信基础：同时持有圣堂通行章和墓园边贸许可，账簿与封印可在两侧市场核验
- 自身利益：获得修车、补水、30日安全通行和文书保管费
- 风险：若伪造、丢失或拒绝交付副本，将同时失去两侧市场资格与商业信用
- 保管方式：丧钟城档案室一份、商队铁匣一份、灰渡市场送达摘要一份
- 中立边界：商队不参加正面守城，但会保护车辆、记录旗号、伤员、签章与货物损失

# 外部承认边界

卷一结束时：

- 圣堂文书只约束一个地方执法体系30日
- 墓园文书只约束一个持牌收尸商及本次索赔30日
- 匪团不承认丧钟城政治地位
- 灰雁商队只证明文书存在与副本未被单方篡改，不赋予主权
- 丧钟城获得的是“能够缔约、保管争议并承担义务”的事实地位，不是完整国际承认
