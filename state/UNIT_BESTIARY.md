# 兵种志与开智个体登记

> 阶位是军事和社会分类，不是属性面板。兵种首次进入正文时，必须登记来源、成长方式、真实成本、能力边界、社会认知和已知个体。

# 一阶：骷髅兵

- 种族：亡灵
- 常见来源：尸骸经死亡魔力与招魂仪式驱动
- 生产条件：合法或非法尸骸、亡灵施法者、死亡环境、修复材料
- 饮食：不喝水，不进食普通粮食
- 常见能力：不需要呼吸和睡眠；疼痛反应弱；可执行简单命令；适合高污染和夜间劳动
- 维护需求：骨粉胶、金属扣件、绑带、防裂油、稳定死亡魔力；觉醒个体还需要姓名与灵魂锚定
- 明确弱点：关节、脊柱和灵魂火焰；低智个体缺乏复杂判断；死亡魔力紊乱时可能失控
- 社会认知：圣堂普遍视为邪恶工具；墓园奴役派视为创造者财产
- 世界真相：少量个体可能恢复或形成连续人格
- 控制边界：控制牌与骨架内的实体锚点可强制低智个体行动；控制者死亡后锚点会衰弱，但不必然立即失效
- 觉醒表现：灵魂火焰稳定后可形成有限语言，不依赖完整声带；持续命名、偏好、主动修补与拒绝命令都是连续人格证据
- 法律区分：
  - 低智骷髅：非法律人格，但受尸骸权与公共安全规则保护
  - 觉醒骷髅：具备连续自我后可申请法律人格和城民身份
- 已知开智个体：鲁尔
- 首次完整开智呈现：第一卷灰脊劳役道；开篇不另设会写名字的骷髅

# 幽魂

- 种族：亡灵灵体
- 常见来源：强烈执念、异常死亡环境、灵魂仪式或墓园生态
- 饮食：不喝水，不进食普通粮食
- 常见能力：穿过部分普通障碍、感知灵魂、夜间侦察、保存部分记忆
- 维护需求：稳定锚点、姓名、记忆载体、适量死亡魔力与不受干扰的档案环境
- 明确弱点：圣光、灵魂封印、姓名与记忆消散、强风或特定魔法场
- 法律人格：是否具备连续自我必须逐个判断，不因幽魂身份自动承认或否认
- 已知个体：塞西娅

# 行尸与低智肉身亡灵

- 种族：亡灵
- 饮食：通常不进食；特殊食尸种另行登记
- 维护需求：防腐盐、缝合材料、油脂、低温或稳定死亡魔力
- 风险：腐败、疫病传播、命令失控、尸骸来源争议
- 法律地位：无连续人格者不是城民，但尸体来源、原主意愿和公共安全必须登记

# 吸血鬼

- 种族：高等亡灵或死亡血脉生命，具体学派认知不同
- 普通饮水：清水不是核心能量来源，但用于诊疗、药剂、清洁和血液保存
- 核心需求：血液或可替代血液制剂
- 维护与伦理：血液来源、同意、价格、医疗优先级必须登记
- 已知个体：伊莱恩

# 天使

- 社会分类：七阶高阶生命
- 常见来源：独立高阶种族；祭坛不制造天使
- 常见能力：飞行、圣力、强大身体、军团威压
- 明确弱点：羽翼受创会失血并失去飞行能力；不能以自身治疗术修复自身；死亡不可逆
- 社会认知：圣堂宣称天使是神圣秩序的绝对证明
- 已知个体：瓦蕾莎

# 八阶堕落天使

- 性质：唯一法则生命，不是可繁育兵种
- 已知个体：瓦蕾莎
- 诞生条件：白骨长峡中返生圣仪、墓园核心、遗愿共鸣、永昼圣印、天使血脉和自主拒绝共同冲突
- 战力定位：正面战力约等于十名普通七阶同类，但核心价值是改变复活与契约法则
- 明确限制：不可量产；过度使用损害存在稳定性；不服从无条件控制
- 当前状态：远期锁定，正文前尚不存在

# 城市互惠规则

不同种族不需要相同资源，但可以依赖同一套城市系统：

- 活人需要水、粮食、医疗与住房
- 骷髅需要躯体维护与稳定死亡魔力
- 幽魂需要姓名、档案与灵魂锚点
- 吸血鬼需要合法血液和诊疗条件
- 所有群体都需要城防、道路、法律、记录和免于被强制控制的空间

公平不等于平均分水，而是公开承认真实需求，并让各群体承担与能力相适应的义务。
