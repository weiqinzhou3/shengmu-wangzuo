# 第一卷第六章状态快照

- 章节：《无主之城》
- 时间：D1日出后
- 地点：北部旧军道旁废弃石灰窑、通往灰脊的废弃劳役道
- 主要视角：沈砚

## 核心事件

- 沈砚在废弃石灰窑确认无明显污染后，为瓦蕾莎清洗伤口、压迫止血并重新固定伤翼。
- 银钉暂不拔除，因为可能压住血管，且缺乏照明、止血药和熟悉羽翼结构的治疗者。
- 瓦蕾莎用“七阶不是不死”明确天使无法自我治疗，较强生命力只意味着更能承受伤势。
- 沈砚结合先遣队资料与处置点战斗，理解一至七阶的质变关系，以及同阶内部仍存在经验、控制和装备差异。
- 正文确认天使不是天生七阶，瓦蕾莎由较低阶位长期成长至资深七阶。
- 白誓不再使用字母等级。它是瓦蕾莎个人长期重铸的武器，能够承受七阶力量，但不会赋予她新的阶位或额外能力。
- 瓦蕾莎说明奥伦在沈砚离开后封锁北仓、焚烧记录，并沿北部道路转移三箱星银。
- 转移单和上级印章只能证明扣押预案提前存在，不能证明签署者、最终目的或正式处决命令。
- 两人正式交换由本人说出的姓名，而非只依赖文书称呼。
- 丧钟城被选择为落脚点，因为那里可能仍有诊所、水井和居民，又不受圣堂稳定控制；本章不披露它是否属于净化行动目标。

## 力量设定

- 一至四阶：依靠身体、血脉、训练与经验积累。
- 五阶：稳定感知并引导环境魔力的门槛。
- 六阶：与环境魔力形成持续共鸣和控制。
- 七阶：能够以个人意志影响局部战场，是当前大陆常规战力顶点。
- 同阶强弱不使用字母面板，通过初入、稳固、资深、巅峰等作者侧模糊判断和实际表现区分。
- 瓦蕾莎当前为资深七阶，但伤势、失血与环境仍能显著限制她。

## 前往丧钟城的动机

- 瓦蕾莎需要正式治疗，沈砚的碱水、绷带和食物即将耗尽。
- 圣堂道路、白灰窑和原接应路线容易被奥伦搜索。
- 丧钟城是最近已知可能存在医生、屋顶、水井和补给的聚居地。
- 丧钟城没有稳定城主，也不受圣堂、墓园或地方领主持续控制。
- 两人不知道诊所是否仍然开放，也不知道当地居民是否会接纳他们。

## 人物与关系状态

- 沈砚：仍处灰髓疫观察期；承认营救同时包含分诊判断、无人救援和瓦蕾莎熟悉道路等现实原因。
- 瓦蕾莎：没有宣誓效忠，也没有把救命债转化为契约；开始认可沈砚不美化自身动机。
- 二人关系：暂时共同求生，先抵达丧钟城，再决定之后的事。
- 爱情状态：无。

## 知识边界

- 当前确认：接应营记录被焚毁；三箱星银沿北路转移；扣押方案提前准备；奥伦本人无权单独签发相关上级文书。
- 当前不能确认：签署者身份、扣押最终目的、河谷与白墙驿完整结果、灰髓疫人为来源、净化行动战争目的、丧钟城是否是后续目标。

## 资源状态

- 沈砚：半瓶以下碱水、少量绷带和食物，折叠刀严重损坏。
- 瓦蕾莎：白誓在身边，剑体完整；无法飞行、独立远行或再次进行同等级爆发。
- 交通：使用从处置点带出的押送车与马匹。

## 章末状态

- 二人驶上通往灰脊的废弃劳役道。
- 丧钟城只是最近可能提供治疗和藏身条件的现实落脚点。
- 奥伦尚未进入本章视角，其后续追索速度未知。
