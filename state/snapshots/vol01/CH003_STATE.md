# 第一卷第三章状态快照

- 章节：《第三声警钟》
- 时间：D0，第三声警钟至入夜
- 地点：接应营、界门坐标架、西侧通道、西沟与排水涵洞
- 主要视角：沈砚

## 核心事件

- 第三声警钟响起，隔离结界破裂。
- 华夏坐标组发现星银与界门坐标架发生异常共振，主队启动紧急撤离。
- 周启明拆下裂界指北针并随主队撤离。
- 奥伦焚烧死亡名单、转运记录和物资文书；沈砚从火中取得瓦蕾莎的转移单。
- 四名裁判骑士使用刻符文的锁链协同控制瓦蕾莎，一名骑士趁她保护平板车时以银钉刺伤左翼根部。
- 奥伦命令将瓦蕾莎活着送往第七静默处置点接受扣押审查。
- 沈砚服从瓦蕾莎“先带最后一辆车离开”的要求，帮助平板车越过西门。
- 沈砚确认车队继续驶向旧白灰窑后，折返寻找华夏主队；坐标锚杆坍塌，他跌入排水涵洞。

## 固定时间线

1. 瓦蕾莎在营内被控制。
2. 沈砚先护送最后一辆车越过西沟。
3. 沈砚折返寻找主队。
4. 沈砚坠入涵洞并失去意识。
5. 瓦蕾莎仍留在营内，之后才被押往第七静默处置点。

## 战力逻辑

- 瓦蕾莎不是被四名骑士正面击败。
- 她已长期阻挡圣焰，周围是难民和伤员，无法释放穿透人群的攻击。
- 她不愿率先杀死仍可能只是服从命令的同袍。
- 奥伦熟悉她的性格，利用平板车、多人配合和翼根偷袭完成控制。
- 银钉破坏羽翼关节和飞行能力，不会夺走七阶力量。

## 人物与资源状态

- 沈砚：右肩在界门冲击与坠落中受伤，额角破裂，防护服损坏，灰髓尘暴露风险增加。
- 瓦蕾莎：左翼根部重伤，明显失血，被锁链控制；白誓被扣押。
- 周启明：持裂界指北针随主队撤离，生存状态正常。
- 奥伦：继续控制接应营、北仓和记录处置。
- 难民车队：至少一辆灰封车和一辆平板车驶出西沟，最终状态未知。

## 知识边界

- 沈砚知道星银与界门事故有关，但不能确认是否为蓄意破坏。
- 沈砚知道瓦蕾莎被送往第七静默处置点，但尚未正式决定营救。
- 瓦蕾莎不知道沈砚坠入涵洞。

## 章末状态

- 沈砚与华夏主队失联，独自困在黑暗涵洞。
- 瓦蕾莎被送往秘密扣押设施。
- 接应营冲突从公开程序争议升级为记录销毁、战略物资异常和高阶人员秘密扣押。
