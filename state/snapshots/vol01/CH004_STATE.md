# 第一卷第四章状态快照

- 章节：《北门之外》
- 时间：D0深夜至后半夜
- 地点：排水涵洞、接应营西北低地、北门、北部旧军道、第七静默处置点外围与维护室
- 主要视角：沈砚

## 核心事件

- 沈砚在第三声警钟后约三个时辰醒来，完成颅脑、四肢和右肩检查，并清点有限物资。
- 转移单显示“拒绝现场处置”是后补内容，而姓名、目的地和押送安排早已填写，证明拘捕方案提前准备。
- 沈砚离开涵洞后焚烧污染外层，以碱水去污，并记录灰髓尘暴露时间与当前症状。
- 正文完整说明灰髓尘传播、早期症状、重症死亡与尸骨再污染风险。
- 沈砚比较三条路线后决定营救瓦蕾莎：主队无法定位；难民车队已有最低保障且自己可能带去污染；瓦蕾莎位置明确、危险迫近、无人救援。
- 沈砚沿北门车辙、暗金血迹、断枝和旧军道路标追踪至第七静默处置点。
- 他确认“静默”只是秘密扣押代称，处置点内外声音正常传播。
- 沈砚从后方检修水沟潜入维护室，发现瓦蕾莎被锁在石椅上，翼根银钉未拔且无人治疗。

## 沈砚动机

- 职业：按分诊原则优先处理位置确定、危险迫近、仍可干预的伤员。
- 道德：瓦蕾莎因保护最后一辆车被捕，他无法明知去向后放弃。
- 求生：她熟悉边境并具有高阶战力，合作比独自流浪更可执行。
- 调查：她掌握奥伦与净化行动信息，可能解释接应营和界门异常。

## 人物与物资状态

- 沈砚：右肩伤势持续，额角与虎口有伤；无明确灰髓疫症状，仍处十八至三十六小时潜伏窗口。
- 物资：折叠刀刃口在撬开检修石板时受损；碱水、绷带、食物与可用布料减少。
- 瓦蕾莎：清醒，明显失血和缺水；双腕、腰腿被锁；左翼被固定，银钉仍在翼根；尚不知道沈砚姓名。

## 知识边界

- 沈砚只能确认拘捕提前准备、扣押中故意不给治疗，不能确认上级人物和最终目的。
- 沈砚尚不知道白誓等级和具体作用。
- 瓦蕾莎不知道沈砚为何来到、也不知道他坠入涵洞后的经历。

## 章末状态

- 沈砚已经进入处置点维护室。
- 瓦蕾莎发现他以后首先低声要求他离开。
- 营救尚未开始，两名守卫仍在场。
