# 第一卷第七章状态快照

## 基础信息

- 章节：第七章《劳役道》
- 日期：D1
- 时段：上午至夜间
- 地点：灰脊劳役道西段、废弃劳役工棚
- 主要视角：沈砚

## 人物状态

### 沈砚

- 右肩伤势在囚车倾斜和拆车时被牵动，仍不能持续发力。
- 食物与可饮用水继续下降；灰髓疫观察期无新增明确症状。
- 观察到鲁尔能够主动整理工具、修正支撑、发现拖架绳结问题并连续自称姓名。
- 尚未宣布鲁尔具备完整法律人格。

### 瓦蕾莎

- D1仍不能独立站立或行走；伤翼保持固定但肿胀未消。
- 可以坐姿使用白誓削木和协助判断承重结构。
- 对鲁尔保持戒备，没有因亡灵身份直接攻击。

### 鲁尔

- 首次进入正文。
- 左前臂使用颜色更深的替换骨，关节由三条皮带固定。
- 持有小石锤、两枚凿子和多块练习姓名的石片。
- 已表现：工具分类、石工判断、主动提醒、连续刻写、自称“鲁尔”。

### 赫斯·卡罗

- D1暮色后从劳役道西侧抵达，已主动说明姓名与持牌收尸人身份。
- 出现原因：原控制者三日未归，雇佣矿场将劳役队编号交给赫斯；他沿西段工棚逐处寻找而来。
- 带有一头驮骡、一辆由两具受控骷髅牵引的空板车、骨架固定箱、控制牌和黄铜回收牌。
- 说明自己负责回收失去控制者的劳役队；因天黑不验骨、不接控制链，决定在工棚外过夜并于天亮核验。
- 已听见鲁尔自称姓名，但尚未判断该行为是否代表连续人格。

## 资源与位置

- 旧囚车：左轮、车轴和铁箍损坏，已经拆解退出使用。
- 新拖架：由车门、长板、横木、窗栏、皮带和毡毯改制。
- 两匹普通马：一匹牵引拖架，一匹驮运与轮换；均未命名。
- 白誓：剑体完整，本章仅作普通切削工具和防卫武器，未发生高强度使用。

- 双方已明确等待天亮核对编号、文书和骨架状态；第八章必须从同一夜或次日天亮连续承接。
