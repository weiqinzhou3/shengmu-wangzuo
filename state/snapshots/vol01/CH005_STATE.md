# 第一卷第五章状态快照

- 章节：《第七静默处置点》
- 时间：D1天亮前约一小时至黎明
- 地点：第七静默处置点维护室、扣押室与石林东侧道路
- 主要视角：沈砚

## 核心事件

- 沈砚观察扣押室、锁链、守卫换位和维护铁轮，拒绝直接冲入。
- 他利用维护铁轮松开瓦蕾莎双腕锁链，行动被守卫察觉。
- 瓦蕾莎用松开的锁链制造短暂机会，沈砚从墙上取下白誓并送回她手中。
- 白誓没有提供外部力量，只帮助瓦蕾莎稳定集中自身七阶圣力。
- 瓦蕾莎先要求两名守卫让开；两人继续攻击后，她以近距离破甲剑术杀死二人。
- 战斗完整表现七阶与普通骑士的实力差距，同时立即撕裂瓦蕾莎伤口、加剧失血。
- 沈砚用守卫钥匙解除其余锁链，不拔除翼根银钉，先做压迫止血和固定。
- 瓦蕾莎明确说明自己不能自我治疗，圣堂治疗需要另一名施术者引导。
- 沈砚带走转移副本、接收记录和带有上级印章的普通授权纸。
- 两人使用原押送车离开处置点。

## 战力与宝物规则

- 瓦蕾莎的力量来自自身，不来自羽翼或白誓。
- 羽翼受伤造成失血、疼痛、平衡和飞行障碍。
- 白誓是她熟悉的高阶武器，本章尚未由她说明A级等级。
- 白誓不记录、不通信、不鉴定、不治疗。
- 处置点两名守卫没有平民掩护，也已明确选择继续攻击，因此接应营的限制不再存在。

## 人物状态

- 沈砚：右肩伤势在撬锁、转移中加重；折叠刀进一步损坏；仍无明确灰髓疫症状。
- 瓦蕾莎：左翼关节重伤，银钉保留；双腕锁伤；七阶短时爆发后无法站立与持续战斗。
- 两名留守骑士：死亡。
- 奥伦：不在处置点，尚不知道守卫已经死亡和瓦蕾莎脱困。

## 关系变化

- 沈砚证明自己不是冲动闯入，而是能够观察和利用设施完成救援。
- 瓦蕾莎重新获得行动选择，但尚未正式得知沈砚姓名。
- 二人仍不是同盟，只因逃离与伤势形成事实上的共同处境。

## 章末状态

- 两人乘押送车离开石林。
- 处置点很快会因换岗发现两具尸体。
- 瓦蕾莎急需医生、药物和稳定落脚点。
