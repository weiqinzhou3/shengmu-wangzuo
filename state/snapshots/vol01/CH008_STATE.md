# 第一卷第八章状态快照

## 基础信息

- 章节：第八章《财产的标准》
- 日期：D2
- 时段：清晨至上午
- 地点：灰脊劳役道废弃工棚、旧装卸台
- 主要视角：沈砚

## 人物状态

### 沈砚

- 没有替鲁尔作法律结论，也无法担保其未来绝不伤人。
- 以鲁尔昨夜的主动行为、当场修理和东行表达质疑普通回收流程。
- 仍不能替丧钟城承诺接纳鲁尔。

### 瓦蕾莎

- 坐姿和重伤状态下仍能以七阶威压迫使赫斯后撤亡灵，并在章末短时站立、压制现场九具骷髅。
- 原本准备允许赫斯回收全部八具；因鲁尔能预见破坏、拒绝命令并照料同伴而改变判断。
- 不替鲁尔担保；若其攻击活人，仍会亲手杀死。
- 不允许赫斯在确认异常前清除鲁尔记忆。

### 鲁尔

- 拒绝佩戴四二一号牌。
- 违抗砸毁石阶的命令，改为修复会牵连工棚立柱的基础。
- 主动指向拖架、瓦蕾莎伤翼和东路，表达同行意愿。
- 章末被旧控制链强迫向西归队，仍抱着姓名石片说“不回”。

### 赫斯·卡罗

- D2清晨从西侧抵达；原领工实际已死多时，矿场直到石料断供才在前一日报失，赫斯为回收和道路安全连夜寻找。
- 到场后先清点八具遗骸、查验领工私印与主牌、核对编号并修复骨架。
- 右手缺失一截小指，源于早年误判一具反复呼喊旧主姓名的失控骷髅。
- 受雇回收劳役队并承担维修、牌照与道路风险，不因鲁尔会说名字便愿意放弃责任。
- 用破损主牌和回收文书测试控制链；旧链被唤醒但未转入其控制。

## 事件状态

- 七具普通劳役骷髅已接受赫斯的新主牌并归入板车队列。
- 鲁尔的四二一号牌尚在赫斯手中，鲁尔拒绝佩戴。
- 旧控制链进入失控激活状态，赫斯无法直接解除。

## 下一章必须承接

- 鲁尔正在被强迫向西移动，身体服从、声音拒绝。
- 瓦蕾莎已短时站立并压制现场亡灵。
- 赫斯必须参与解除自己触发的危机，不能继续机械清点或重新开始辩论。
